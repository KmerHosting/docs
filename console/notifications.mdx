## Available actions

- Open the notifications page to read the full list.
- Mark a notification as read when it has been reviewed.
- Delete a notification that is no longer needed.
- Use the unread summary in the console navigation for quick awareness.

## Email preferences

Notification records in the console are separate from email-delivery preferences.
