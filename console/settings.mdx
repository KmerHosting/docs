The Settings page currently exposes notification preferences such as invoice emails, service-expiry messages, security alerts, and newsletters.

## Recommended configuration

Keep invoice, expiry, and security notifications enabled for operational accounts. Newsletter preference can be selected independently.

## Saving changes

The page loads settings from the authenticated user endpoint and saves the complete preference state. Wait for the success confirmation before leaving the page.
