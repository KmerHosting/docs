## Current limits and behavior

- Total quota: 5 GB per account.
- Maximum individual file size: 1 GB.
- Files larger than 5 MB use chunked upload in the web interface.
- The file list supports download and typed-confirmation deletion.
- Transfers can use a new upload or an existing file in the library.
- The receiver activates temporary visibility for 5, 15, 30, 60, or 120 minutes.
- The sender selects an online visible user and sends the file.

## Storage and KmerHosting Drive

KmerHosting Drive is the product-facing name for the storage experience. Free Storage describes the included account entitlement and current quota.
