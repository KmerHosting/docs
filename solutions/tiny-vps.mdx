Tiny VPS is KmerHosting's entry-level virtual server solution. It provides an isolated Linux environment with SSH and root access while using fewer resources than a full VPS.

## Best suited for

- Small APIs and backend services
- Development and staging environments
- Cron jobs and automation workers
- Private tools and internal dashboards
- Lightweight websites and applications
- Linux experiments and learning environments

## Included with every plan

- LXC virtualization
- One snapshot
- One free `.com` domain
- Shared IPv4 and IPv6
- Linux operating system
- 200 Mbit/s network port
- Unlimited traffic
- Full SSH access
- Root access

## Tiny VPS compared with VPS

Tiny VPS is intended for compact workloads and lower resource usage.

Choose VPS Hosting when you require:

- KVM virtualization
- Dedicated IPv4
- More CPU and RAM
- Larger storage
- Higher network capacity
- Sustained production workloads
