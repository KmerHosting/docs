The KmerHosting SSL Interface aggregates domains from Shared Hosting, Tiny VPS, VPS, Custom Hosting, and standalone domain records.

## What the current interface does

- Lists domains attached to the authenticated account.
- Identifies the service source for each domain.
- Presents the certificate state as active in the current console view.
- Provides a shortcut to open the HTTPS website.

## Operational interpretation

Certificate issuance and renewal depend on correct DNS, reachable hosting, and the actual Let’s Encrypt or hosting-panel configuration. The console list is not a complete certificate diagnostic tool.

<Warning>
Do not treat the green state alone as proof that every browser and hostname is correctly serving the expected certificate. Open the site and inspect the certificate when troubleshooting.
</Warning>
