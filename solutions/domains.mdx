KmerHosting lets you search, buy, renew, and manage domains from the customer area.

## Common extensions

| Extension | Recommended use |
|---|---|
| `.com` | Global standard. |
| `.org` | Organizations and communities. |
| `.net` | Network or technical projects. |
| `.cm` | Local Cameroon presence. |
| `.info` | Information websites. |
| `.biz` | Commercial activities. |
| `.site` | General-purpose websites. |
| `.online` | Online presence. |
| `.store` | E-commerce. |
| `.app` | Applications. |
| `.dev` | Developers. |
| `.ai` | Artificial intelligence and machine learning. |

<Note>
Prices change depending on the registrar, active promotions, and billing term. Always check the Pricing page before purchase.
</Note>

## Domain types

- **Standalone domain**: purchased and managed independently from a hosting service.
- **Included domain**: attached to a specific offer or hosting service.

## DNS

Common DNS records are:

| Type | Usage |
|---|---|
| `A` | Points a name to an IPv4 address. |
| `AAAA` | Points a name to an IPv6 address. |
| `CNAME` | Creates an alias to another hostname. |
| `MX` | Configures incoming mail delivery. |
| `TXT` | Verification, SPF, DKIM, and DMARC. |
| `NS` | Defines the authoritative nameservers. |

<Warning>
DNS changes take time to propagate. Avoid changing several critical records at the same time.
</Warning>
