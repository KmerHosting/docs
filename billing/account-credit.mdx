## Start a top-up

1. Open Billing.
2. Select Add Credit.
3. Enter the amount requested in the wallet context.
4. Continue to CamerPay.
5. Choose an available payment method.
6. Complete the hosted payment flow.
7. Return to KmerHosting and wait for confirmed crediting.

## Payment safety

Do not repeat a payment simply because the browser return is delayed. Check the wallet and transaction state first, then contact support with the provider reference if required.
