KmerHosting billing is built around four main objects: cart, order, invoice, and account credit.

## Standard flow

```text
Cart -> Order -> Payment -> Paid invoice -> Activation or renewal
```

## Important objects

| Object | Role |
|---|---|
| Cart | Holds services before checkout. |
| Order | Represents the purchase request. |
| Invoice | Payment document and accounting record. |
| Account credit | Available balance used for orders or renewals. |

## Currency

This documentation uses FCFA as the main example currency. Exact amounts should always be confirmed in the customer area.

<Warning>
Never pay an invoice without confirming the amount, status, and the official payment method shown by KmerHosting.
</Warning>
