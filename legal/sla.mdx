Cette page décrit les principes de disponibilité et de support. Les engagements précis doivent être validés dans le document SLA officiel.

## Document officiel

Consultez la version officielle : `https://kmerhosting.com/service-level-agreement`.

## Avant d’ouvrir une demande

Préparez :

- votre email de compte ;
- l’ID de commande ou facture ;
- le service concerné ;
- la date de l’événement ;
- les preuves utiles sans mot de passe ni secret.

<Warning>
En cas de différence entre cette page documentaire et le document légal officiel, le document officiel prévaut.
</Warning>
