KmerHosting Drive is the product identity for the account storage interface. The current implementation is available in the console Storage section.

<Frame caption="KmerHosting Drive currently uses the same interface as the 5 GB Free Storage workspace.">
  <img src="/images/screenshots/drive-overview.png" alt="KmerHosting Drive currently uses the same interface as the 5 GB Free Storage workspace." />
</Frame>


## Current functions

- Upload files up to 1 GB each.
- Monitor the 5 GB account quota.
- Download and delete stored files.
- Become temporarily visible to receive a transfer.
- Send a new or existing file to another visible KmerHosting user.
