KmerHosting permet de rechercher, acheter, renouveler et gérer des domaines depuis l’espace client.

## Extensions courantes

| Extension | Usage recommandé |
|---|---|
| `.com` | Standard mondial. |
| `.org` | Organisations et communautés. |
| `.net` | Projets réseau ou techniques. |
| `.cm` | Présence locale camerounaise. |
| `.info` | Sites d’information. |
| `.biz` | Activités commerciales. |
| `.site` | Sites polyvalents. |
| `.online` | Présence en ligne. |
| `.store` | E-commerce. |
| `.app` | Applications. |
| `.dev` | Développeurs. |
| `.ai` | Intelligence artificielle et machine learning. |

<Note>
Les prix changent selon le registrar, la promotion et la période. Vérifiez toujours la page Pricing avant achat.
</Note>

## Types de domaines

- **Domaine standalone** : acheté et géré indépendamment d’un service d’hébergement.
- **Domaine inclus** : rattaché à une offre ou à un service spécifique.

## DNS

Les enregistrements DNS courants sont :

| Type | Usage |
|---|---|
| `A` | Pointe un nom vers une IPv4. |
| `AAAA` | Pointe un nom vers une IPv6. |
| `CNAME` | Crée un alias vers un autre nom. |
| `MX` | Configure la réception email. |
| `TXT` | Vérifications, SPF, DKIM, DMARC. |
| `NS` | Définit les serveurs de noms. |

<Warning>
Une modification DNS peut prendre du temps à se propager. Évitez de changer plusieurs paramètres critiques en même temps.
</Warning>
