## Current capabilities

- Send a message to the assistant.
- Save conversations to the authenticated account.
- List saved conversations.
- Delete a conversation owned by the account.
- Use the API-key protected AI Agent endpoints through the SDK.

## Data safety

Never submit root passwords, API keys, KYC documents, payment secrets, production database dumps, or unreviewed proprietary source code.
