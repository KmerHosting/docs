## Common codes

| Code | Meaning | Action |
|---:|---|---|
| `200` | Successful response | Read the returned JSON. |
| `201` | Resource created | Store the returned identifier. |
| `400` | Invalid request | Check the body, parameters, and format. |
| `401` | Authentication required | Sign in or refresh the session. |
| `404` | Resource not found | Verify the ID and the URL. |
| `500` | Server error | Retry, then contact support if it persists. |

## Diagnosis

Include these details in an API ticket:

- endpoint called;
- HTTP method;
- response code;
- response excerpt without secrets;
- timestamp;
- account or order ID when applicable.
