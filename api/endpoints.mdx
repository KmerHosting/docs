All account automation endpoints use the `/api/v1/sdk` prefix.

| Resource | Capabilities |
|---|---|
| Account | Read profile, account status, wallet currency, credit, and debt |
| Services | List and inspect Shared hosting, Custom Hosting, VPS, and Tiny VPS |
| Auto-renewal | Enable or disable service auto-renewal |
| Domains | List standalone and service-associated domains |
| Billing | List issued and upcoming invoices |
| Orders | List orders and line items |
| Notifications | List, mark, and delete notifications |
| KmerHosting AI Agent | Stream messages and manage owned conversations |

See the generated API reference for request parameters, response schemas, scopes, and examples.
