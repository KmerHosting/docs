This page summarizes endpoint families. The detailed API reference is generated from `openapi.json`.

## Public

| Method | Endpoint | Usage |
|---|---|---|
| `GET` | `/api/v1/content/hero` | Localized public content. |
| `GET` | `/api/v1/catalog/offers` | Offers available during checkout. |
| `GET` | `/api/v1/catalog/trending-services` | Highlighted services. |
| `POST` | `/api/v1/auth/register` | Account creation. |
| `POST` | `/api/v1/newsletter/subscriptions` | Newsletter subscription. |

## Authenticated user

| Method | Endpoint | Usage |
|---|---|---|
| `GET` | `/api/v1/users/me/profile` | Account profile. |
| `GET` | `/api/v1/cart` | Current cart. |
| `POST` | `/api/v1/orders` | Order creation. |
| `GET` | `/api/v1/invoices` | Invoice list. |
| `GET` | `/api/v1/users/me/services/active` | Active services. |
| `POST` | `/api/v1/domains/availability-checks` | Domain availability lookup. |
