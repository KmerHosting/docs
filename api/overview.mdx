The KmerHosting API exposes public endpoints and user endpoints under the `/api/v1` namespace.

## Base URL

```text
https://kmerhosting.com
```

## Main categories

- Authentication
- Catalog
- Cart
- Orders
- Billing
- Invoices
- Domains
- Services
- KYC
- Notifications
- Storage
- Users

## Authentication

Public endpoints do not require a session. User endpoints rely on the signed-in account session through cookies.
