L’API KmerHosting expose des endpoints publics et des endpoints utilisateur sous l’espace `/api/v1`.

## Base URL

```text
https://kmerhosting.com
```

## Catégories principales

- Authentication.
- Catalog.
- Cart.
- Orders.
- Billing.
- Invoices.
- Domains.
- Services.
- KYC.
- Notifications.
- Storage.
- Users.

## Authentification

Les endpoints publics ne nécessitent pas de session. Les endpoints utilisateur utilisent la session du compte connecté via cookie.
