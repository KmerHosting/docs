---
title: "API authentication"
description: "Authenticate Developer API requests with a KmerHosting API key."
icon: "key"
---

Create an API key from **Customer console → Account → API Keys**. The complete value is displayed once and is never stored in plaintext by KmerHosting.

```http
Authorization: Bearer kh_live_...
```

Example:

```bash
curl https://kmerhosting.com/api/v1/sdk/account \
  -H "Authorization: Bearer $KMERHOSTING_API_KEY"
```

<Warning>
Keep API keys in server-side secret storage. Never place a key in browser code, a public mobile application, source control, screenshots, or logs.
</Warning>

Keys are account-scoped, can be deactivated or deleted at any time, and are checked against the scope required by each endpoint.
