Email verification confirms that you control the address used for the KmerHosting account.

## How it works

After registration, KmerHosting sends a verification link or token to the account email address. Open the link from the same browser when possible.

## Common issues

| Problem | Solution |
|---|---|
| Email not received | Check spam, promotions, and quarantine folders. |
| Link expired | Request a new verification link. |
| Invalid token | Copy the full link or restart the process. |
| Wrong address | Contact support if you can no longer access the account. |

## After verification

Once the email is verified, you can access the customer area, follow orders, and receive service notifications.
