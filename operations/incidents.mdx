Un incident est une interruption, dégradation ou erreur affectant un service.

## Cycle d’incident

| Statut | Signification |
|---|---|
| Investigating | Le problème est en cours d’analyse. |
| Identified | La cause probable est identifiée. |
| Monitoring | Un correctif est appliqué et surveillé. |
| Resolved | Le service est revenu à la normale. |

## Modèle de communication

```text
Incident détecté : nous investiguons un problème affectant [service].
Cause identifiée : un correctif est en cours.
Monitoring : le correctif est appliqué, nous surveillons la stabilité.
Résolu : le service fonctionne à nouveau normalement.
```
