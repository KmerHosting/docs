An incident is an outage, degradation, or error affecting a service.

## Incident lifecycle

| Status | Meaning |
|---|---|
| Investigating | The issue is currently being analyzed. |
| Identified | The likely cause has been identified. |
| Monitoring | A fix has been applied and is being watched. |
| Resolved | The service has returned to normal. |

## Communication template

```text
Incident detected: we are investigating an issue affecting [service].
Cause identified: a fix is currently being applied.
Monitoring: the fix is in place and we are monitoring stability.
Resolved: the service is operating normally again.
```
