## Production base URL

```text
https://kmerhosting.com/api/v1/sdk
```

## Authentication

Every Developer API request uses a KmerHosting API key:

```bash
curl https://kmerhosting.com/api/v1/sdk/account \
  -H "Authorization: Bearer $KMERHOSTING_API_KEY"
```

Create and revoke keys from **Console → Account → API Keys**.

<Warning>
Never place an API key in browser JavaScript, a public mobile bundle, a Git repository, logs, screenshots, or client-visible environment variables. Use the SDK from a trusted server, worker, CI job, or local terminal.
</Warning>

## Available resources

- account and billing overview;
- Shared hosting, Custom Hosting, VPS, and Tiny VPS;
- service expiration and auto-renewal settings;
- standalone and service-linked domains;
- issued and upcoming invoices;
- orders;
- notifications;
- KmerHosting AI Agent messages and saved conversations.

The API never returns hosting passwords, VPS root passwords, KYC documents, session cookies, provider secrets, or administrative resources.
