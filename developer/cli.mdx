## Install

```bash
npm install --global @kmerhosting/cli
```

## Configure

```bash
export KMERHOSTING_API_KEY="kh_live_..."
```

On PowerShell:

```powershell
$env:KMERHOSTING_API_KEY="kh_live_..."
```

## Commands

```bash
kh account
kh services
kh services vps
kh domains
kh invoices
kh orders
kh notifications --unread
kh auto-renew vps SERVICE_ID on
```
