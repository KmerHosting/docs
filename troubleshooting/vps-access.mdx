## Checks

- VPS IP address.
- SSH port.
- Correct SSH key.
- Local and server firewall rules.
- VPS status in the customer area.

## Useful commands

```bash
ssh -v root@YOUR_VPS_IP
ping YOUR_VPS_IP
nc -vz YOUR_VPS_IP 22
```

<Warning>
Never share a private SSH key or a root password in a support ticket.
</Warning>
