## Causes possibles

- Domaine mal pointé.
- SSL non actif.
- Fichier d’accueil absent.
- Erreur PHP ou WordPress.
- Quota de stockage atteint.
- Base de données inaccessible.

## Informations utiles

- Domaine.
- URL exacte.
- Message d’erreur.
- Dernière modification effectuée.
- Capture d’écran.
