An SSL certificate allows your domain to serve traffic over HTTPS.

## Requirements

- The domain must point correctly to the service.
- Ports 80 and 443 must be reachable for a VPS.
- DNS must be propagated.

## Common issues

| Problem | Possible cause |
|---|---|
| Validation fails | DNS not propagated or domain not pointing correctly. |
| HTTPS unavailable | Certificate not installed or server not configured correctly. |
| Mixed content | HTTP resources still loaded inside an HTTPS page. |

<Warning>
On VPS, SSL depends on your server configuration. On shared hosting, use the control panel or contact support.
</Warning>
