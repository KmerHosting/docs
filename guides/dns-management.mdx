DNS records control where a domain points: website, email, verification, and technical routing.

## Add a record

<Steps>
  <Step title="Open the domain">
    From the customer area, select the standalone or included domain.
  </Step>
  <Step title="Open DNS records">
    Click the DNS management or zone settings section.
  </Step>
  <Step title="Choose the type">
    Select `A`, `CNAME`, `TXT`, `MX`, or another supported record type.
  </Step>
  <Step title="Enter the value">
    Copy the exact value provided by the third-party service or by KmerHosting.
  </Step>
  <Step title="Save and wait">
    Propagation can take from a few minutes to several hours.
  </Step>
</Steps>

## Examples

| Need | Type | Example |
|---|---|---|
| Website to server | `A` | `@ → 203.0.113.10` |
| Subdomain to service | `CNAME` | `www → example.com` |
| Domain verification | `TXT` | Value provided by the third-party service |
| Incoming email | `MX` | Mail server provided |

<Warning>
Removing or changing a DNS record can interrupt the website or email flow.
</Warning>
