## Include

- Account email or customer ID.
- Service type and service ID.
- Domain or hostname.
- Order, invoice, or transaction reference when relevant.
- Exact timestamp and timezone.
- Expected result and actual result.
- Reproduction steps.
- A sanitized screenshot or log excerpt.

## Never include

- Passwords.
- API keys.
- Session cookies.
- KYC documents.
- Private keys.
- Complete database dumps.
