Domain availability is checked through the KmerHosting domain search and API workflow.

## Search

Use a complete domain name such as `example.com`. The result can include availability and current price-book information.

## Purchase path

1. Select an available domain.
2. Add it to the cart or service configuration.
3. Review the registration term and price.
4. Create the order.
5. Complete payment before order expiry.
6. Wait for the domain to appear under standalone or included domains.

## Important timing

Availability can change between search and completed registration. An unpaid cart or pending order does not guarantee final registration.
