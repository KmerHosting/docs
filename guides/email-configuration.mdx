Professional email usually depends on specific DNS records.

## Common records

| Type | Role |
|---|---|
| `MX` | Tells the domain which servers receive email. |
| `SPF` | Authorizes sending servers. |
| `DKIM` | Signs outgoing mail. |
| `DMARC` | Defines the anti-spoofing policy. |

## Best practices

- Configure SPF, DKIM, and DMARC.
- Use strong passwords for mailboxes.
- Avoid unsolicited bulk sending.
- Monitor deliverability errors.
