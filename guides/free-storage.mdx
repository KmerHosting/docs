The Storage page is available to every authenticated KmerHosting account and currently provides a 5 GB quota.

<Frame caption="The overview displays quota usage and the file library.">
  <img src="/images/screenshots/storage-overview.png" alt="The overview displays quota usage and the file library." />
</Frame>


## Upload a file

1. Select Upload.
2. Choose or drag a file.
3. Confirm the file is no larger than 1 GB.
4. Wait for the progress indicator to complete.
5. Confirm the file appears in the library.

<Frame caption="Large files are uploaded in chunks by the web interface.">
  <img src="/images/screenshots/storage-upload.png" alt="Large files are uploaded in chunks by the web interface." />
</Frame>


## Receive a file

Activate temporary visibility and select a duration: 5, 15, 30, 60, or 120 minutes.

<Frame caption="Visibility should be enabled only for the time required to receive the transfer.">
  <img src="/images/screenshots/storage-visibility.png" alt="Visibility should be enabled only for the time required to receive the transfer." />
</Frame>


The sender can then find the visible test account.

<Frame caption="The recipient list shows only accounts that temporarily enabled visibility.">
  <img src="/images/screenshots/storage-recipient-list.png" alt="The recipient list shows only accounts that temporarily enabled visibility." />
</Frame>


## Send a file

Choose the recipient, then upload a new file or select an existing library item.

<Frame caption="Confirm the recipient before starting the transfer.">
  <img src="/images/screenshots/storage-transfer.png" alt="Confirm the recipient before starting the transfer." />
</Frame>


Received files are marked in the library.

<Frame caption="The Received badge distinguishes transferred files from normal uploads.">
  <img src="/images/screenshots/storage-received-file.png" alt="The Received badge distinguishes transferred files from normal uploads." />
</Frame>


## Delete a file

Deletion requires a typed confirmation phrase.

<Frame caption="Typed confirmation reduces accidental permanent deletion.">
  <img src="/images/screenshots/storage-delete-confirmation.png" alt="Typed confirmation reduces accidental permanent deletion." />
</Frame>


<Warning>
Storage is not a replacement for a tested multi-copy backup strategy. Keep critical data in an independent backup location.
</Warning>
