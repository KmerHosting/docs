Les services actifs sont visibles depuis l’espace client. Selon le type de service, vous pouvez consulter les paramètres, renouveler, gérer le domaine ou demander une action support.

## Actions courantes

| Action | Où la faire |
|---|---|
| Voir les détails | Fiche du service dans l’espace client. |
| Renouveler | Section renouvellement ou commande. |
| Gérer le domaine | Page domaines ou service associé. |
| Voir les paramètres serveur | Fiche du service ou email d’activation. |
| Demander une modification | Ticket support. |

<Note>
Pour les services sensibles, certaines actions peuvent nécessiter une vérification du compte.
</Note>
