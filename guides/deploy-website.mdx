Le processus dépend du type de service : Shared hosting, VPS ou custom hosting.

## Shared hosting

<Steps>
  <Step title="Récupérer les accès">
    Ouvrez la fiche du service ou l’email d’activation.
  </Step>
  <Step title="Uploader les fichiers">
    Utilisez le gestionnaire de fichiers ou FTP/SFTP selon le panneau.
  </Step>
  <Step title="Configurer la base de données">
    Créez la base et l’utilisateur si votre application en a besoin.
  </Step>
  <Step title="Pointer le domaine">
    Configurez les DNS vers l’hébergement.
  </Step>
  <Step title="Activer SSL">
    Installez ou activez le certificat SSL.
  </Step>
</Steps>

## VPS

Sur VPS, installez le serveur web, configurez le firewall, déployez l’application et configurez le reverse proxy ou le service web.
